# tabata-core-privacy

Public privacy policy for **Tabata Core** (Google Play). Static site: `index.html` only.

## Enable GitHub Pages

1. On GitHub: **Settings → Pages → Build and deployment → Source**: *Deploy from a branch*.
2. Branch **main**, folder **/ (root)**.
3. Your policy URL will be: `https://tasosstyl1984.github.io/tabata-core-privacy/`

Contact email is set in `index.html`. Update it there if it changes, then push to `main` so GitHub Pages stays current.

Keep in sync with `TabataCore/tabata-core-privacy/index.html` when you change policy text.

## First-time push (after you create the empty repo on GitHub)

Create a **new public** repository named `tabata-core-privacy` under `tasosstyl1984` (no README, no .gitignore).

Then run:

```bash
cd "c:\Users\tasos\OneDrive\Documents\GitHub\tabata-core-privacy"
git remote add origin https://github.com/tasosstyl1984/tabata-core-privacy.git
git push -u origin main
```

If GitHub shows HTTPS auth issues, use Git Credential Manager or a [personal access token](https://github.com/settings/tokens).
